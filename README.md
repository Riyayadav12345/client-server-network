# 🖧 Client-Server Network (Cisco Packet Tracer)

## 📌 About
Basic client-server network built using Cisco Packet Tracer.

## 🖥️ Devices Used
- 3x PC (Clients)
- 1x Server
- 1x Switch (2960-24TT)

## 🌐 IP Configuration
| Device  | IP Address    | Subnet Mask   |
|---------|--------------|---------------|
| PC1     | 192.168.10.2 | 255.255.255.0 |
| PC2     | 192.168.10.3 | 255.255.255.0 |
| PC3     | 192.168.10.4 | 255.255.255.0 |
| Server1 | 192.168.10.1 | 255.255.255.0 |

## ✅ Results
- Ping test: 0% packet loss
- Web server accessible from all clients
